Welcome to Zanata

Zanata is a web-based system for Translators to translate documentation and software online using a web browser.

Please join the mailing list to be kept informed of outages and other announcements.

Signing In

Users with a Fedora Account System (FAS) account can use their FAS accounts to authenticate to this server. You can also use an OpenID (eg from Google or myOpenID), or create a local account. You can attach multiple identities to a single account from your Profile page.

Please make sure you join the mailing list .

Translators

All you need is a FAS account and a web browser to start translating. To get started, please follow the simple instructions in the Translator Guide or the Translator Quickstart to join your language team and start contributing translations to the various projects on Zanata .

Developers and Writers

With Zanata you can open your project for translation without opening up the project's version control. Currently, Zanata supports translation of software and documentation projects that provide GNU Gettext format Portable Object (PO) files or Java Properties files. Project files can be pushed and pulled from the server using a Maven plugin or a Python client.

For detailed information on how to manage your projects on Zanata , please read the Quickstart Guide and other documentation .

Create a project


To create a translation project in Zanata for your open source software or documentation project, please obtain a FAS account and validate it with Zanata ; then visit the Projects page and click " Create project " (on the right).

Know More

You can find out more about the open source Zanata project (mailing lists, bug reports, source code) at http://zanata.org/
