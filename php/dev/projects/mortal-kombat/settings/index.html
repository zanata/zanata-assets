    Copy Trans

    Copy Trans attempts to reuse translations that have been entered in Zanata by matching them with unstranslated strings in your project/version. Consequently, Copy Trans is best used before translation and review work is initiated on a project.

    Each translation has to pass through each of these checks before it will be copied.

    1. On content mismatch

      If the translations are not identical

    2. On context mismatch

      If the context (resId, msgctxt) of the translations are not identical

    3. On project mismatch

      If the translations are not both from this project

    4. On document mismatch

      If the translations are not both from the same document & document path

    5. If all previous steps have passed, copy as translated.

      Unless previously marked as fuzzy.
